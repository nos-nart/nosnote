### ðŸ¦¡ Mongoose timestamp

Mongoose schemas have a `timestamps` option that tells Mongoose to automatically manage `createdAt` and `updatedAt` properties on documents

<CodeBlock language="js">{`
const userSchema = mongoose.Schema({
  email: String
}, { timestamps: true });
const User = mongoose.model('User', userSchema);
const doc = await User.create({ email: 'test@google.com' });
doc.createdAt; // 2020-07-06T20:36:59.414Z
doc.updatedAt; // 2020-07-06T20:36:59.414Z
doc.createdAt instanceof Date; // true
`}</CodeBlock>
